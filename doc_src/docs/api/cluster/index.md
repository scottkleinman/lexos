# Cluster

The `cluster` module provides methods for generating and visualizing cluster analyses. The [`dendrogram`](dendrogram.md) module performs hierarchical agglomerative clustering, and the [`kmeans`](kmeans.md) module perfors k-means clustering.

The [`plotly_dendrogram`](plotly_dendrogram.md) module performs the same function as [`dendrogram`](dendrogram.md) but uses the Python <a href="https://plotly.github.io/plotly.py-docs/index.html" target="_blank">Plotly</a> library to generate dendrograms.
